Flask To-Do List

  • The user login details and tasks are stored in the database with the help of the FlaskSQLAlchemy library.

Flask Weather App

A simple weather app built using Flask and Bootstrap.

  • The app lets you enter the name of any city around the world and get the weather info of that city.
  • I used Open Weather Map API for weather data.
  • The requests library fetches the data from the API. This data is then used by Flask to update the weather info on the frontend.

Random Wiki Article App

A simple app that displays the title and introduction of a random wikipedia article when you click the buuton and gives you a link to the article just in case you want to read it the whole thing.

  • I used the Beautiful Soup library to scrape the Title and Introduction of the wikipedia article.
  • I used flask to create a JSON reponse which is then used by the javascript fetch api to load the article title and info without reloading the whole page.
